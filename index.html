Manviny.fileinfo

Download this project as a .zip file Download this project as a tar.gz file

Guarda información sobre un fichero (imagen, audio...)

La información se guarda en:

  • dynamodb
  • mysql
  • mongodb *json

La información puede ser:

  • path del fichero
  • nombre del fichero
  • descripción (idiomas)
  • etiquetas
  • lat/lng
